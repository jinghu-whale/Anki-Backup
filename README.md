# Anki-Backup
My anki cards' backups, which have no sheduling informations.

# Info

Your can open apkg file by [anki client](https://apps.ankiweb.net/).

* **collection-[date]@[time].colpkg**

  Anki will copy your entire collection into a file ending in .colpkg, and place it on your desktop. A collection package is used to back up your collection, or copy it to another device.

* **All Decks.apkg**

  `All Decks.apkg` contains below all apkg files' content.

* **algorithm.apkg**

  `algorithm.apkg` contains algorithms from leetcode, newcower and so on.

* **backend-interviewer.apkg**

  `backend-interviewer.apkg` contains backend interviewer questions and answer.
